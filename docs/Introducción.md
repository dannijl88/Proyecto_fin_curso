# 1. Introducción

## Objetivo del proyecto

Desarrollar una tienda online completa de venta de velas artesanales con panel de administración. El sistema permite gestionar productos, procesar pedidos y administrar usuarios con diferentes roles.

## Problema que resuelve

Pequeñas empresas de velas artesanales necesitan una plataforma accesible para vender sus productos online sin depender de marketplaces externos. La solución ofrece:

- Catálogo digital de productos con stock actualizado
- Carrito de compras y procesamiento de pedidos
- Panel de control para gestionar inventario y ventas
- Sistema de autenticación para clientes y administradores

## Contexto

Proyecto final del ciclo formativo de Desarrollo de Aplicaciones Web (DAW). Demuestra competencias en desarrollo backend, frontend, bases de datos y despliegue en producción.
